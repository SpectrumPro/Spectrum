## Object-oriented lighting control


> **Beta Software Notice!**  
> Spectrum is currently in **active development** and is considered **beta software**.  
> Features may change, bugs may exist, and stability is **not yet guaranteed**.  
> It is **not recommended for mission-critical or production use** at this stage.


**Spectrum** is a modern, object-oriented lighting control platform designed to replace unintuitive rigid workflows with a flexible system that scales from simple setups to highly complex productions.


## Key Features

- **Client & Server Support**  
  Run Spectrum in distributed environments, separating control, visualization, and output across multiple devices.

- **Peer-to-Peer Network Engine**  
  A decentralized networking model that enables reliable and low-latency communication without a single point of failure.

- **GDTF Fixture Importing**  
  Import industry-standard GDTF fixture definitions for accurate and consistent fixture behavior.

- **Fully Modular User Interface**  
  A flexible, module-based UI that adapts to different workflows and use cases.

- **Cross-Platform**  
  Runs on major desktop platforms, ensuring consistent behavior across operating systems.
